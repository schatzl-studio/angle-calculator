# angle-calculator
A simple web-based trigonometric calculator

---

## Features

- Calculates all missing values of a right-angled triangle  
- Flexible input combinations:
  - two sides  
  - one side + one angle  
  - two angles  
- Automatic validation and consistency checks  
- Dynamic SVG visualization  
- Mirror and rotate view  
- Clear distinction between user input and calculated values  
- Mobile-optimized layout  
- No dependencies (single HTML file)

---

## Geometric Model

- γ = 90°  
- a – opposite α  
- b – opposite β  
- c – hypotenuse  
- α + β = 90°

Calculations are based on the Pythagorean theorem and trigonometric functions.

---

## Usage

Open `angle-calculator.html` in any contemporary browser.  
No installation required.
